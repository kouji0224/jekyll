## 1.rbenvとruby-buildのアップデート

rbenv

```
$ cd ~/.rbenv
$ git pull
```

ruby-build

```
$ cd ~/.rbenv/plugins/ruby-build
$ git pull
```

## 2.インストールできるRubyのバージョン確認

```
$ rbenv install -l
```

## 3.Ruby2.7のインストール

```
$ rbenv install 2.7.0
```

## 4.Rubyのバージョン切り替え

```
$ rbenv global 2.7.0
```

これでOK！簡単だぜ！
